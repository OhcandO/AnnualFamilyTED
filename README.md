## 미팅(이벤트) 컨셉 정리

### 한 줄 요약
> *"자주 만나지만 서로의 세계는 잘 모르는 가족"* 을 위한,
> 한 사람당 **15분의 무대** 위에서 각자의 영역을 제대로 소개하는 첫 자리.

### Why — 왜 이 모임을 하는가
- 우리는 자주 모이지만, 서로가 어떤 일에 **열정을 쏟고 있는지** 깊이 들여다볼 기회는 적었다.
- 새 가족, 새 커리어, 깊어진 취미 — 변화한 서로의 세계를 **판을 깔고, 마이크를 켜고** 제대로 들어보자.
- 익숙한 얼굴의 **낯선 면**을 발견하는 하루.

## 페이지 구성 (`docs/index.html`)

GitHub Pages의 단일 진입점. 넷플릭스풍 다크 히어로에서 시작해 스크롤할수록
새벽처럼 밝아지는 원페이지로, 의존성 없는 순수 HTML/CSS/JS입니다.

1. **Hero** (다크) — 타이틀 · 일시 · 장소 배지
2. **초대의 글** — 모임의 취지
3. **TED 소개** — TED가 처음인 가족을 위한 설명 + 우리 가족 버전 규칙
4. **하루의 흐름** — 점심 → 파티룸 이동 → 본행사 → 다과 → 종료
5. **장소** — TBA · 파티룸 시설 일러스트(SVG)
6. **드레스 코드** — 주황 포인트 아이템 예시(SVG)
7. **클로징**

설계 원칙:
- 노안 배려 — 큰 글자(기본 18px+)·고대비·두꺼운 한글 폰트(Black Han Sans + Noto Sans KR)
- 모바일(갤럭시/아이폰) 우선 레이아웃, 가로 스크롤 없음
- 빌드 도구·프레임워크 없음 — 단일 HTML 파일
